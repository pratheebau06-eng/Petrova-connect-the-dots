# PETROVA - Career Simulation Platform

## Aim
To help students understand different careers by giving them real-world like experiences.

## Objective
To create a system where users can try different career roles and get suggestions based on their performance.

## Description
PETROVA is a career simulation platform that helps students choose the right career. Instead of guessing or following others, users can try out different job roles through simple simulations.

The system gives tasks related to each career. Based on how the user performs, it analyzes their skills like thinking, creativity, and problem-solving, and suggests suitable career options.

This helps students make better decisions and reduces confusion while choosing a career.

## Features
- Try different career roles  
- Perform simple tasks  
- Get basic performance analysis  
- Career suggestions based on skills  

## Technologies Used
- Python  
- Basic AI concepts  

## How it Works
1. User selects a career  
2. System gives related tasks  
3. User completes the tasks  
4. System checks performance  
5. Career suggestions are given  

## Future Scope
- Add more career options  
- Improve AI accuracy  
- Connect with real job platforms
